Aufgaben für Bedingungen und Schleifen

Sortierverfahren

Bubblesort:

Zuerst muss der Text in den ASCII Code übersetzt werden, das hilft uns, die Buchstaben in einer Reihenfolge zu bestimmen. Nachdem der Text in den ASCII Code übersetzt wurde, kann man sie nun miteinander vergleichen. Wenn einige nicht übereinstimmen, dann muss man die Buchstaben vertauschen, sodass sie dann in der richtigen Reihenfolge sind.

Selectionsort:

Beim Selectionsort gibt es einen Teil, der sortiert ist und einen Teil, der unsortiert ist. Es wird immer das kleinste Element genommen und an den Anfang der Liste geschoben. Dies geht immer so weiter, bis es in einer Reihenfolge ist.

Radix Sort

Beim Radixsort wird der Text zunächst in den ASCII Code gebracht, danach wird die Reihenfolge der Ziffern sortiert. Als erstes werden die Einerzahlen sortiert, dann die Zehnerzahlen und dann die Hunderterzahlen.

BucketSort

Beim Bucket Sort wird das Array in mehrere Gruppen, sogenannte "Buckets", aufgeteilt. Jeder Wert wird in den passenden Bucket sortiert. Danach wird jeder Bucket für sich selbst sortiert, oft mit einer einfachen Methode. Am Ende werden alle sortierten Buckets zusammengefügt, und das Array ist vollständig sortiert. Dieser Algorithmus ist besonders schnell, wenn die Werte gleichmäßig verteilt sind.
